# Zutaten:
- {% ingredient 125 %}g Udon Nudeln (vorgekocht oder frisch)
- {% ingredient 150 %}ml Kokosmilch
- {% ingredient 50 %}ml Wasser
- {% ingredient 50 %}g Brokkoli
- {% ingredient 50 %}g Blattspinat
- {% ingredient 0.5 %} Möhren
- {% ingredient 0.5 %} Lauchzwiebeln
- 1 EL Gochujang (koreanische Chilipaste)
- 1 EL Ahornsirup/Honig
- 1 EL Erdnussbutter
- 2 EL Sojasauce
- Sojasprossen

# Rezept
1. {% ingredient 125 %}g Udon Nudeln in eine große Pfanne oder einen Wok geben.
2. {% ingredient 0.5 %} Möhren in Streifen hobeln und {% ingredient 0.5 %} Lauchzwiebeln in Ringe schneiden
2. {% ingredient 0.5 %} Möhren, {% ingredient 50 %}g Brokkoli, {% ingredient 50 %}g Blattspinat, {% ingredient 0.5 %} Lauchzwiebeln und Sojasprossen darüber verteilen.
3. Für die Sauce 1 EL Gochujang, 1 EL Ahornsirup/Honig, 1 EL Erdnussbutter, 2 EL Sojasauce, {% ingredient 150 %}ml Kokosmilch und {% ingredient 50 %}ml Wasser verrühren und über das Gemüse gießen.
4. Deckel auf den Topf/die Pfanne setzen und bei mittlerer Hitze alles ca. 10 – 12 Minuten einköcheln lassen, bis das Gemüse bissfest und die Sauce cremig ist.
